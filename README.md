# gias-transformation-prototype
Alpha prototypes for testing iterations of GIAS with users.

## Version 0.1

Included MVP content for establishments and only two options for filtering to test users response to what they needed to be there.

## Version 0.2

Fixed some issues with the search.

### Home page
Added a link to the closed establishments archive.
 
### Search results page

Removed “*” from results page when selecting “View all establishments”
 
#### Change search

Change or clear the search term in the search field and hit return or press the [Search] button to apply new search and keep any already selected filters.
 
#### Trust types

Split Trust Type into:
- Trust: Multi-academy Trust
- Trust: Single-academy Trust

#### Closed establishments

Removed all closed establishments from the general search results and moved them to a closed establishments archive.
 
#### Status

Removed ‘Status’ from filters and results as we have open and closed separate. Status still shown on individual establishment and establishment group pages.
 
### Establishment page
 
- Changed “Last updated…” to “Data verified” tag. “Last updated…” date could be 3 years ago but the school could have checked and verified it this month. “Data verified” keeps it simple.
- Added a download link for establishment data.
- Added 'History' section that shows for any establishments that have a previous iteration.
 
### Establishment group page
 
- Changed “Last updated…” to “Not updated since…” tag. “Last updated…” date could be 3 years ago but the group could have checked and verified it this month. “Not updated since…” shows that it could be out-of-date.
- Member establishments listed underneath. i.e. academies listed if establishment group is a trust. 
- Added a download link

### Archive of closed establishments

New page added for viewing and searching closed establishments.
